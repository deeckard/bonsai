## Installation

- Open the Extensions sidebar in VS Code
- Search for Bonsai
- Click on Install
- Open the Command Palette with `Ctrl + Shift + P` or `⇧ ⌘ P`
- Select `Preferences: Color Theme` and choose the Bonsai color scheme
- Enjoy!
